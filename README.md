# Fake Ecommerce Store

A simple e-commerce website built with JavaScript that uses a fake API to perform complete CRUD operations. The application allows users to view, create, update, and delete products through API requests.

## Features

- View all products from a fake API
- Add new products
- Edit existing products
- Delete products
- View product details
- Dynamic product cards
- Form handling for creating and updating products
- Responsive layout

## CRUD Operations

### GET
Fetches and displays products from the API and renders them on the website.

### POST
Creates and adds new products to the store through a form submission.

### PUT
Updates existing product information by editing product details and sending the changes to the API.

### DELETE
Removes products from the store by sending a delete request to the API.

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- Fake REST API

## How It Works

The website communicates with a fake API endpoint to manage product data. JavaScript handles all API requests and updates the user interface dynamically.

Users can:

1. View available products.
2. Add new products.
3. Edit product information.
4. Delete products.

## Project Purpose

This project was created to practice working with REST APIs and understand how CRUD operations are implemented in a real-world e-commerce application.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/harlardex/Fake-ecommerce-web.git
```

2. Open the project folder.

3. Launch the project using a live server or open the `index.html` file in your browser.
